Static Vector Search Demo

Science article search with client-side embeddings

This demo loads a JSON file, reads precomputed embeddings, creates a query embedding in the browser, and ranks articles with cosine similarity. No backend or database is required.

Examples:

How it works

  1. The app fetches data/articles.json.
  2. Each article already includes an embedding vector.
  3. A lightweight keyword encoder turns your query into a vector.
  4. Cosine similarity ranks the most relevant articles.
